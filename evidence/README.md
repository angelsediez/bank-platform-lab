# Evidence

This directory stores evidence collected while building and operating the lab.

Evidence should be practical and reviewable. It is meant to capture validation results, screenshots, and drill notes that help document what was tested and what was observed.

## Structure

- `screenshots/` for visual evidence
- `validation-results/` for command outputs and checks
- `drills/` for recovery and operational exercise notes

## Rules

- keep filenames consistent
- capture evidence after validation succeeds
- prefer operationally meaningful artifacts over decorative screenshots
