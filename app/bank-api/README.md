# bank-api

This directory contains the Go-based banking API used as the main demo workload for the lab.

The service is expected to remain small and operationally focused. It exists to support platform practice, not to become a large product by itself.

## Expected responsibilities

- account-related endpoints
- transfer-related endpoints
- database integration
- health and readiness endpoints
- metrics, traces, and structured logs
- container build and deployment packaging
