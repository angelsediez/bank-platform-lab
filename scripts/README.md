# Scripts

This directory contains helper scripts used to support repository maintenance, infrastructure work, cluster bootstrap, validation, and operational tasks.

Scripts in this repository should stay small, focused, and easy to review.

## Structure

- `dev/` for local repository helpers
- `infra/` for infrastructure-related helpers
- `cluster/` for cluster bootstrap helpers
- `ops/` for operational validation helpers
- `backup/` for backup and restore support helpers
